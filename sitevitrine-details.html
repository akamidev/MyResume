Site Vitrine JavaScript - Gestion dynamique et responsive

Ce projet a été conçu dans le cadre d'un TP de JavaScript, avec pour objectif la création d'un site vitrine interactif utilisant exclusivement du JavaScript Vanilla. L'accent a été mis sur la manipulation du DOM, la création d'un CRUD complet et l'intégration de comportements dynamiques sans framework.

Parmi les fonctionnalités principales, le site inclut un système de facturation CRUD (Création, Lecture, Mise à jour, Suppression), un menu burger pour les écrans mobiles, des onglets interactifs sur la page d'accueil, une modification dynamique des tarifs, ainsi qu'un filtre de produits selon leur catégorie.

Ce projet m'a permis de renforcer mes compétences en JavaScript pur et de comprendre les bases de l'interactivité sur un site web statique.

Mehdi AKAMI

Intégrateur Web

Le projet est totalement responsive et adapté à tous les types d'écrans. Sa structure modulaire permet une évolution facile et une personnalisation rapide.

Un exemple concret de ce qu'il est possible de réaliser avec HTML, CSS et JavaScript, sans bibliothèques tierces, dans une logique de formation et de mise en pratique de compétences fondamentales.
