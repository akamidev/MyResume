Kasa – Application de location d’appartements

Kasa est une plateforme web réactive développée en React, permettant aux utilisateurs de parcourir des logements à louer. L’interface a été intégrée à partir d’un design Figma et alimentée par une API de logements.

Chaque logement dispose d'une fiche détaillée avec galerie d’images, tags, équipement, hôte, et note. Le site comprend aussi une gestion des erreurs avec une page 404 animée.

Ce projet m’a permis d’approfondir mes compétences en React, gestion des routes, composants dynamiques, et intégration d’une API.

Mehdi AKAMI

Développeur Front-End React

Le projet est entièrement responsive, modulable, avec code propre basé sur les composants, SCSS et modules CSS. Une référence pour des applications modernes à fort contenu visuel.
