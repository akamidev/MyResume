Nettoyage Automatique des Fichiers Temporaires

Ce projet Python supprime automatiquement les fichiers temporaires stockés dans les répertoires système de Windows, comme %TEMP%. Il permet de maintenir un système propre et fluide, sans intervention manuelle.

  • ✅ Nettoyage automatique des caches système
  • ✅ Compatible avec le Planificateur de tâches
  • ✅ Facilement personnalisable (ajout de dossiers)
  • ✅ Exécutable via ligne de commande ou script .bat

Un script utile que j’utilise personnellement pour garder mes systèmes Windows propres. Il est simple, rapide, et personnalisable.
